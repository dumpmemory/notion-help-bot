# Notion-Help-Bot

A Telegram bot designed for [Notion CN Community](https://t.me/Notionso).
Powered by [Notion Official API](https://developers.notion.com/)

# Features

- Notion as a **dataase** 
- Keyword triggers
- Keyword mentioned counts
- Useful tools recommendation
